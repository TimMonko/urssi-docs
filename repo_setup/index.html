Github Repo Documentation That Isn't a Website¶

  1. README.md -- Also where CONTRIBUTING.md information can live
  2. LICENSE -- use MIT or BSD3, generally
  3. CODE_OF_CONDUCT.md -- use Contributor Covenant with this markdown text
  4. CITATION.cff -- recognized by Github, some Citation Managers, and Zenodo -- the latter of which is a great way to get a DOI for your repo!
  5. Update pyproject.toml
